# ⚡ Physics Bot Module & Dataset Mirror (`physics_bot/`)

This directory contains the Physics subject module, dataset mirror, and web asset copy for the **GPT Physics Bot (`@gptphysicsbot`)**.

---

## 📂 File & Directory Breakdown

```
physics_bot/
├── data/
│   └── questions_dataset.json   # 5,000 Physics MCQs across 20 chapters
├── web/
│   ├── index.html               # Web interface copy
│   ├── style.css                # CSS design system copy
│   ├── app.js                   # Client JS copy
│   └── logo.png                 # Logo asset copy
└── README.md                    # Module documentation
```

---

## 📄 File Details

- **`data/questions_dataset.json`**: Standalone mirror containing 5,000 Physics MCQs (250 questions per chapter) across all 20 NTA Physics chapters (*Units & Measurements, Mechanics, Gravitation, Thermodynamics, Electrodynamics, Optics, Modern Physics, Semiconductors*).
- **`web/`**: Web asset directory for Physics-specific deployments.
