# 📦 Scraped Data & Raw PYQ Archive (`data/`)

This directory contains raw scraped NEET previous year questions (PYQs), NCERT paragraph mappings, and historical question archives used during initial dataset construction.

---

## 📂 Subdirectory Breakdown

| Subdirectory | Contents & Purpose |
| :--- | :--- |
| **`pyq_analysis_bio/`** | Raw scraped NEET Biology PYQ topic weightage analysis files and paragraph-level NCERT question mappings. |
| **`bio_ncert/`** | Raw NCERT Class 11 and Class 12 Biology paragraph text extracts and question pairings. |
| **`questions/`** | Raw scraped JSON question batches harvested from educational portals. |
| **`pyqs/`** | Raw PDF and JSON archives of authentic NEET 2016-2026 examination papers. |
