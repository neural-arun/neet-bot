# 🧪 Chemistry Bot Module & Dataset Mirror (`chemistry_bot/`)

This directory contains the Chemistry subject module, dataset mirror, and web asset copy for the **GPT Chemistry Bot (`@gptchemistrybot`)**.

---

## 📂 File & Directory Breakdown

```
chemistry_bot/
├── data/
│   └── questions_dataset.json   # 5,000 Chemistry MCQs across 20 chapters
├── web/
│   ├── index.html               # Web interface copy
│   ├── style.css                # CSS design system copy
│   ├── app.js                   # Client JS copy
│   └── logo.png                 # Logo asset copy
└── README.md                    # Module documentation
```

---

## 📄 File Details

- **`data/questions_dataset.json`**: Standalone mirror containing 5,000 Chemistry MCQs (250 questions per chapter) across all 20 NTA Chemistry chapters (*Some Basic Concepts, Atomic Structure, Bonding, Thermodynamics, Equilibrium, Kinetics, Organic Principles, Hydrocarbons, Amines, Biomolecules*).
- **`web/`**: Web asset directory for Chemistry-specific deployments.
